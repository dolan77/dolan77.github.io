Feast Review

Feast Review is an application that provides more of a user feedback experience for people who want to eat at restaurants. This project aims to help users determine which locations are better to eat at than others. The project utilizes firebase to store user data to a database and the Yelp API to retrieve already mapped out restauarnts around the globe.

This project is currently a work in progress
