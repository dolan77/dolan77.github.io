HoloTracker

HoloTracker is a project that I created that consumes a Web API utilizing the ASP.NET MVC pattern. The project utilizes the HoloDex API found here. The project sends a query to return Vtubers from Hololive, Nijisanji, or Independent Streamers that are currently streaming or planning to stream within the next 48 hours.

Because of my inexperience, I am unable to upload the project on GitHub Pages due to my use of the ASP.NET MVC pattern. There will be plans on uploading the project to a website but if you would like to check out the project, please run it locally on your machine through an IDE (Visual Studio).

Check it out here!

Mega Wars is a project that takes a nice spin on the classic card game: Wars. This version of wars allows more than two players to play with more than one deck, allowing for a single game to run many iterations until there is a winner. The project was done in C++.

Check it out here!
Feast Review

Feast Review is an application that provides more of a user feedback experience for people who want to eat at restaurants. The project aims to help users determine which locations are better to eat at than others. The project utilizes firebase to store user data to a database and the Yelp API to retrieve already mapped out restauarnts around the globe. The team consists of Nathan Lai, Matthew Hirai, Kenny Du, and I and was completed in the span of six months.

Check it out here!
